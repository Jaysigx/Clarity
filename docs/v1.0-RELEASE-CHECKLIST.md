# Clarity IDE v1.0 Release Checklist

> Target: **First stable release - "The IDE that sees your code"**

## Core Philosophy
- **Zero accounts** - No signup required
- **Zero payment** - Fully free and open source
- **Fully local** - AI runs on your machine or free APIs
- **AI-native** - Built from ground up for AI assistance

---

## What's Done ✅

### Editor (Core Experience)
- [x] Dual-layer editor (textarea + syntax highlighting)
- [x] **Complete syntax highlighting** - ALL tokens visible:
  - Keywords (purple), Strings (green), Numbers (orange)
  - Functions (blue), Classes (yellow), Variables (red/pink)
  - Operators/Punctuation (light blue), Comments (muted)
  - Language-specific: JS/TS, Python, CSS, JSON, Markdown
- [x] Proper scrolling (horizontal + vertical)
- [x] Line numbers with sync scroll
- [x] Auto-indent, bracket matching
- [x] Find & Replace (Ctrl+F / Ctrl+H)
- [x] Go to Line (Ctrl+G)
- [x] Multi-tab with dirty indicators
- [x] Auto-save on change

### AI Panel (The Differentiator)
- [x] **AI Vision** - AI sees your current editor view 👁️
- [x] File attachment (full file context)
- [x] Selection attachment (highlighted code)
- [x] Visual attachment chips with animations
- [x] Streaming chat with all providers
- [x] Composer (AI patch generation)
- [x] Context ranking (BM25)
- [x] Session management

### File Management
- [x] Explorer with 40+ file icons
- [x] New/Rename/Delete files
- [x] Right-click context menu
- [x] Filter/search files

### Git Integration
- [x] Full git status (staged/changes/untracked)
- [x] Diff viewer inline
- [x] Stage/Unstage/Discard
- [x] Commit with message
- [x] Push/Pull/Fetch
- [x] Branch switcher
- [x] **Stash UI** (stash / pop)
- [x] Commit log

### Desktop App
- [x] Frameless window with custom titlebar
- [x] Window controls (min/max/close)
- [x] Native menu bar (File, Edit, View, etc.)
- [x] Draggable topbar
- [x] Panel options (⋯ menu on each panel)

### Settings
- [x] Settings overlay with search
- [x] Theme selection (dark variants)
- [x] Accent colors
- [x] Font size/family
- [x] Word wrap, line numbers
- [x] AI provider configuration
